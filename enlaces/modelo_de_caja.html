En esta pagina, cuando se pincha sobre algunos enlaces, el navegador abandona el sitio web para acceder a paginas que se encuentran en otros sitios. Estos enlaces se conocen como "enlace externos".Sin embargo, la mayoria de enlaces de un sitio web apuntan a paginas del propio sitio web, por lo que se denominan "enlaces internos". Cuando el navegador encuentra un URL relativa que solo consiste en el nombre de un recurso, supone que el protocolo, servidos y directorio del recurso enlazado son los mismos que los del origen del enlace.
